# Цель:
Приобретение навыков решения задач поиска оптимального решения, используя генетический алгоритм.

# Задачи:
* Решить экономическую задачу генетическим алгоритмом
* Сравнить точность и время решения задачи с другим приближенным алгоритмом

# Решение:
1. В качестве задачи была выбрана симметричная Задача коммивояжера.
2. В качестве альтернативного алгоритма был взят Жадный алгоритм
3. Алгоритмы были сравнены по точности, времени и памяти.

# Выводы:
* Жадный алгоритм наиболее простой в реализации, он быстро работает, но не всегда дает наилучший результат.
* Генетический алгоритм намного сложнее в реализации, он состоить из нескольких этапов, требудет подбора параметров. Однако затраты на него могут способствовать наилучшим результатам.
* Генетический алгоритм более точный, но он требует значительно больше памяти и времени работы.
